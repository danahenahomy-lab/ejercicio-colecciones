# Ejercicio: Colecciones de Datos en Java

## 📌 Descripción

Este programa permite registrar estudiantes y sus notas utilizando una colección de datos tipo **HashMap** en Java.

## ⚙️ Funcionalidades

* Agregar estudiantes con su nota
* Mostrar lista de estudiantes
* Buscar estudiante
* Eliminar estudiante

## 🧠 Estructura utilizada

Se utilizó un **HashMap**, que permite almacenar datos en pares:

* Clave: nombre del estudiante
* Valor: nota

## ▶️ Ejecución

1. Compilar:

```
javac EjercicioColecciones.java
```

2. Ejecutar:

```
java EjercicioColecciones
```

## 📷 Ejemplo de uso

El programa muestra un menú donde el usuario puede elegir opciones para gestionar estudiantes.
